屬性經驗值增減:
火屬性育成 -> 火+ 水- 木-
水屬性育成 -> 火- 水+ 木-
木屬性育成 -> 火- 水- 木+
光屬性育成 -> 光+ 闇-
闇屬性育成 -> 闇+ 光-

屬性相剋倍數:
火 -> 木 -> 水 -> 火 (1.5)
火 -> 水 -> 木 -> 火 (0.8)
光 -> 闇 -> 光(1.5)
同屬性 (1)

ex:
我方(火光) -> 敵方(木闇)
火 -> 木 (1.5)
火 -> 闇 (1)
光 -> 木 (1)
1.5 * 1 * 1 = 1.5 (傷害倍數)

我方(火光) <- 敵方(木闇)
火 <- 木 (0.8)
光 -> 木 (1)
火 -> 闇 (1)
0.8 * 1 * 1 = 0.8 (傷害倍數)
